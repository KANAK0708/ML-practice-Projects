# Used Car Price Prediction

## Overview
This project predicts the selling price of used cars using Machine Learning.

## Algorithms Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- Random Forest Regressor

## Techniques Used
- Data Cleaning
- Missing Value Handling
- EDA
- Feature Engineering
- Outlier Treatment
- One Hot Encoding
- Feature Scaling
- Cross Validation
- Hyperparameter Tuning

## Evaluation Metrics
- MAE
- MSE
- RMSE
- R² Score
